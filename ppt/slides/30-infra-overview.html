CHAPTER 05 · INFRASTRUCTURE

每一条缝都抠过：基建的六个模块

V4 的 Section 3 是整篇报告里 infra 味最浓的部分。DeepSeek 把训练和推理的每一个工程缝隙都抠了一遍——下面是这份「工程 checklist」的六个入口。
01
COMMUNICATION
MoE 通信
计算重叠
把 MoE 层拆成四段、用 wave 调度做细粒度流水。理论 1.92× 加速，RL rollout 实测最高 1.96×。
02
DSL
TileLang
领域专用语言
用 TileLang 融合内核，Host Codegen 把 CPU 校验从几十微秒压到 1 微秒。SMT 求解器辅助整数分析。
03
DETERMINISM
批次无关
确定性 kernel
训练与推理比特级一致。双 kernel 策略+独立累加 buffer，弃用 cuBLAS 换 DeepGEMM。
04
QUANTIZATION
FP4 量化
感知训练
MoE 权重和 indexer QK 全上 FP4。FP4→FP8 反量化无损，top-k 提速 2×，KV recall 99.7%。
05
TRAIN FRAMEWORK
训练框架
三板斧
Muon × ZeRO 用 knapsack 分桶，mHC wall-time 开销压到 6.7%，tensor 级 checkpointing 自动重计算。
06
INFERENCE FRAMEWORK
推理框架
KV cache 异构管理
State Cache + KV Cache 两级池。把 shared-prefix KV 落盘，给 agent 长上下文拉低首 token 延迟。
